# AI 提示词管理器

一款兼容 VS Code 架构（Cursor、Copilot Chat、Trae、Windsurf 等）的 IDE 扩展插件。在侧边栏管理提示词，点击即可一键注入当前 AI 对话输入框。

## 功能特性

- **侧边栏面板** — 在活动栏中直接浏览和管理提示词
- **一键注入** — 点击提示词卡片即可注入到当前 AI 对话输入框
- **分类组织** — 将提示词按可折叠的文件夹分类管理
- **拖拽排序** — 拖拽提示词和分类即可调整顺序（搜索状态下自动禁用）
- **搜索与过滤** — 实时搜索并高亮关键词；双击分类名可按分类筛选
- **增删改查** — 通过弹窗新增、编辑、删除提示词和分类
- **导入 / 导出** — 以 JSON 文件备份或分享提示词库
- **内置默认数据** — 预装 12 条常用提示词，涵盖 5 个分类（代码审查、测试、解释代码、重构优化、通用）
- **剪贴板兜底** — 当 DOM 注入不可用时（如 Trae、Windsurf），自动复制到剪贴板并提示粘贴

## 安装

### 从 VSIX 安装（手动）

1. 从 Releases 下载最新的 `.vsix` 文件
2. 打开 VS Code / Cursor
3. 在命令面板中运行 `Extensions: Install from VSIX...`（`Ctrl+Shift+P`）
4. 选择下载的 `.vsix` 文件

### 从源码构建

```bash
git clone <repo-url>
cd Prompt-Management
npm install
npm run build
# 在 VS Code 中按 F5 启动扩展开发宿主
```

## 使用方法

1. 点击活动栏（左侧边栏）中的**聊天气泡图标**打开面板
2. 按分类浏览提示词 — 单击分类标题可折叠/展开
3. **双击**分类标题可按该分类筛选
4. **单击**提示词卡片即可注入到当前 AI 对话输入框
5. 使用底部工具栏新增提示词、分类，或导入/导出

### 快捷键

- `Escape` — 关闭弹窗
- `Enter` — 在提示词/分类弹窗中保存（textarea 内除外）

### 拖拽排序

- 拖拽提示词卡片的 **⠿ 把手**或卡片本身，可在同分类内或跨分类调序
- 拖拽分类标题可调整分类顺序
- 拖拽时会出现**蓝色指示线**标示插入位置
- 搜索/过滤状态下**自动禁用**拖拽，避免排序混乱

## 兼容性

| IDE | 注入方式 |
|-----|---------|
| Cursor | DOM 注入（优先），剪贴板兜底 |
| VS Code + Copilot Chat | DOM 注入（优先），剪贴板兜底 |
| Trae / Windsurf / 其他分支 | 剪贴板兜底 |

## 数据存储

提示词存储在 VS Code 的 `globalState` 中，跨会话持久化。使用**导出**创建 JSON 备份，使用**导入**恢复或分享。

## 开发

```bash
npm install
npm run watch    # 开发模式（带 sourcemap）
npm run build    # 生产构建
npm run lint     # 仅类型检查
```

在 VS Code 中按 **F5** 启动扩展开发宿主窗口进行调试。
